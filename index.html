About ParDF

ParDF is a Linux PDF Editor developed for the #MilliTeknolojiHamlesi and #TEKNOFEST PARDUS APPLICATION DEVELOPMENT initiatives. It was built from scratch to address a perceived long-standing need for a native PDF editor within the Pardus ecosystem, but works on other Linux distributions too.

Developed by word-sys | Barın Güzeldemirci.

The goal is to provide essential PDF text editing capabilities in a straightforward desktop application using modern GTK4 and the LibAdwaita library.

Key Features (Release 1)

  • View PDF documents page by page.
  • Thumbnail sidebar for quick page navigation.
  • Select and edit existing text blocks.
  • Add new text blocks to a page.
  • Adjust font size and color for modified text.
  • Save changes (via Export function due to bug - see below).
  • Export document content as plain text (.txt).
  • Export document as DOCX (requires LibreOffice installed).

How to Install & Run (From Source)

As this is distributed as source code, you need to set up the environment first.

1. Install System Dependencies

Ensure you have the necessary system packages. On Debian, Ubuntu, or Pardus:

sudo apt update
sudo apt install python3-pip python3-venv python3-all-dev build-essential cmake libcairo2-dev libgirepository-2.0-dev python3-gi python3-gi-cairo gir1.2-gtk-4.0 gir1.2-adw-1

(Note: python3-gi, gir1.2-* packages provide GTK/Adwaita support)

2. Download and Extract

Download the .tar.xz or .zip archive from the Download section and extract it.

# Example for tar.xz
tar -xf pardf-main.tar.xz
cd pardf-main/
# Example for zip
unzip pardf-main.zip
cd pardf-main/

(Adjust cd pardf-main/ if the extracted folder has a different name).

3. Set up Python Environment & Install Libraries

It's highly recommended to use a Python virtual environment:

# Navigate into the extracted project directory (if not already there)
# cd pardf-main/

# Create a virtual environment named 'venv'
python3 -m venv venv

# Activate the virtual environment
source venv/bin/activate

# Install required Python packages using pip
pip install pygobject pillow pymupdf numpy

(Remember to activate the environment with source venv/bin/activate every time you want to run ParDF from a new terminal).

4. Run the Application

While the virtual environment is active, run the editor:

python3 run-editor.py

Congratulations, the application should now be running!

! Known Issues (Release 1) !

This is the first release and currently has a known bug:

The main 'Save' button gives an error after editing a PDF.
Work is underway to fix this.

Workaround: To save your changes, please use the application menu (☰ icon, often top-right) and select "Export As..." -> "PDF". This will correctly save your modified document to a new file.
