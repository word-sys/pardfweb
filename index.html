About ParPDF

Pardus PDF Editor is a straightforward desktop application built with GTK4 and Adwaita library, designed primarily for Linux distributions like Pardus, Ubuntu, Debian, and others.

It allows you to open and view PDF documents, select existing text blocks, modify their content, and even add new text elements to the page. While not a full-featured PDF editor, it focuses on making basic text alterations simple and accessible.

Key Features

  • View PDF documents page by page.
  • Thumbnail sidebar for easy page navigation.
  • Select and edit existing text blocks directly on the page.
  • Add new text blocks anywhere on the page.
  • Control font size and color for edited/added text.
  • Save changes back to the PDF (incrementally or as a new file).
  • Export document content as plain text (.txt).
  • Export document as DOCX (requires LibreOffice installed).
  • Modern GTK4/Libadwaita interface.

Dependencies & How to Run

Since this is a source code distribution, you need to install the required libraries first.

Required Dependencies

You will need the following installed on your system:

  • Python 3 (version 3.8 or newer recommended).
  • GTK4 and LibAdwaita 1 GObject Introspection bindings (gir1.2-gtk-4.0, gir1.2-adw-1 on Debian/Ubuntu).
  • Python GObject bindings (python3-gi, python3-gi-cairo on Debian/Ubuntu).
  • Python NumPy library (python3-numpy on Debian/Ubuntu, or via pip).
  • Python PyMuPDF library (python3-pymupdf via pip - recommended installation method).
  • (Optional) LibreOffice for DOCX export (libreoffice-common or similar on Debian/Ubuntu).

Installation Example (Debian/Ubuntu/Pardus):

sudo apt update
# Install system libraries for GTK/Adwaita/GObject & optionally LibreOffice/NumPy
sudo apt install python3 python3-pip python3-gi python3-gi-cairo gir1.2-gtk-4.0 gir1.2-adw-1 python3-numpy libreoffice-common

# Install PyMuPDF using pip (recommended)
sudo pip3 install pymupdf

Note: You might prefer installing Python libraries in a virtual environment instead of globally with sudo pip3.

Running from Source

1. Download either the .zip or .tar.xz archive and extract it.

# Example for tar.xz
tar -xf pardus-pdf-editor-v0.1.0.tar.xz
cd pardus-pdf-editor-v0.1.0/ # Or the directory name created
# Example for zip
unzip pardus-pdf-editor-v0.1.0.zip
cd pardus-pdf-editor-v0.1.0/ # Or the directory name created

2. Make sure all dependencies listed above are installed.

3. Navigate into the extracted directory in your terminal.

4. Run the application using the main script:

python3 run-editor.py

Or, if you made the script executable:

chmod +x run-editor.py
./run-editor.py
