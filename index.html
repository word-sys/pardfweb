Nasıl Kurulur?

Otomatik Kurulum (Önerilen)

Aşağıdaki 'İndirmeler' bölümünden veya GitHub Releases sayfasından pardus-pdf-editor_0.1.0-1_all.deb dosyasını indirin.

Ardından terminali açıp, dosyanın indirildiği dizinde şu komutu çalıştırın:

sudo apt install ./pardus-pdf-editor_0.1.0-1_all.deb

(Eğer bağımlılık hatası alırsanız, sudo apt --fix-broken install komutunu deneyin.)

Manuel Kurulum

  1. GitHub Releases bölümünden son sürüm .zip veya .tar.xz dosyasını indirin.
  2. İndirdiğiniz arşivi (.zip veya .tar.xz) bir klasöre çıkartın.
  3. Sisteminizde gerekli paketlerin kurulu olduğundan emin olun. Terminal'e şu komutu yazarak kurabilirsiniz: 
    sudo apt update
sudo apt install python3-pip python3-venv python3-all-dev build-essential cmake libcairo2-dev libgirepository1.0-dev

    (Not: gir1.2-gtk-4.0, gir1.2-adw-1 gibi paketler genellikle masaüstü ortamıyla birlikte gelir, ancak eksikse kurulmalıdır.)

  4. Terminali, dosyaları çıkarttığınız klasörde açın.
  5. Bir sanal ortam (virtual environment) oluşturun ve aktifleştirin: 
    python3 -m venv venv
source venv/bin/activate
  6. Gerekli Python kütüphanelerini kurun: 
    pip install PyMuPDF numpy

    (Not: python3-gi ve python3-gi-cairo sistem paketleri aracılığıyla kurulduğu için burada pip ile kurulmaz.)

  7. Uygulamayı çalıştırın: 
    python3 run-editor.py

Bilinen Problemler! (MEVCUT HATALAR)

Kaydetme Butonu Hatası: PDF'i düzenlemeden sonra ana 'Kaydet' butonu şu anda düzgün çalışmıyor ve hata verebiliyor. Bu sorunu düzeltmek için çalışıyoruz.

Geçici Çözüm: Dosyayı kaydetmek için lütfen üst menüdeki Seçenekler (3 Çizgi) bölümünden 'PDF'i Dışa Aktar' (Export As PDF) seçeneğini kullanın. Bu yöntemle düzenlenmiş PDF'inizi kaydedebilirsiniz.
