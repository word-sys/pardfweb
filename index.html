ParDF / Pardus PDF Düzenleyicisi

Linux için Açık Kaynak, Basit PDF Metin Düzenleyici

Geliştirici: word-sys (Barın Güzeldemirci)

Hakkında

Milli Teknoloji Hamlesi ve TEKNOFEST ruhuyla Pardus için geliştirilen ParDF, PDF dosyalarındaki metin içeriğini doğrudan düzenlemenizi sağlayan temel bir araçtır. Kullanıcı dostu arayüzü ile Linux ortamında PDF düzenlemeyi kolaylaştırmayı hedefler.

Temel Özellikler: PDF açma/görüntüleme, mevcut metin bloklarını seçme ve düzenleme, yeni metin ekleme, temel metin formatlama (font, boyut, renk - *yeni metinler için*), PDF olarak kaydetme, DOCX/TXT formatında dışa aktarma.

Nasıl Kurulur?

Otomatik Kurulum (Önerilen)

Aşağıdaki 'İndirmeler' bölümünden veya GitHub Releases sayfasından _all.deb uzantılı en son dosyayı indirin.

Terminali açıp, dosyanın indirildiği dizinde şu komutu çalıştırın (Dosya adını indirdiğinizle değiştirin):

sudo apt install ./pardus-pdf-editor_*.deb

(Eğer bağımlılık hatası alırsanız, sudo apt --fix-broken install komutunu deneyin.)

Manuel Kurulum (Geliştiriciler İçin)

  1. GitHub Releases bölümünden son sürüm kaynak kodunu (.zip veya .tar.xz) indirin ve bir klasöre çıkartın.
  2. Gerekli sistem bağımlılıklarını kurun: 
    sudo apt update && sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-4.0 gir1.2-adw-1 python3-numpy python3-pip python3-venv python3-dev libcairo2-dev build-essential libreoffice-common fonts-dejavu-core
    (libreoffice-common DOCX dışa aktarma, fonts-dejavu-core yazı tipi desteği için önerilir.)
  3. Terminali, kaynak kod dosyalarını çıkarttığınız klasörde açın.
  4. Sanal ortam oluşturup aktifleştirin: 
    python3 -m venv venv
source venv/bin/activate
  5. Gerekli Python kütüphanesini kurun: 
    pip install wheel PyMuPDF
    (wheel PyMuPDF kurulumu için gerekebilir.)
  6. Uygulamayı çalıştırın: 
    python3 run-editor.py
