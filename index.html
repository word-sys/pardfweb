Nasıl Kurulur?

Otomatik Kurulum (Önerilen)

Aşağıdaki 'İndirmeler' bölümünden veya GitHub Releases sayfasından _all.deb uzantılı dosyayı indirin.

Ardından terminali açıp, dosyanın indirildiği dizinde şu komutu çalıştırın (örnek dosya adı kullanılmıştır, indirdiğiniz dosya adını yazın):

sudo apt install ./pardus-pdf-editor_*.deb

(Eğer bağımlılık hatası alırsanız, sudo apt --fix-broken install komutunu deneyin.)

Manuel Kurulum (Geliştiriciler İçin)

  1. GitHub Releases bölümünden son sürüm kaynak kodunu (.zip veya .tar.xz) indirin.
  2. İndirdiğiniz arşivi bir klasöre çıkartın.
  3. Gerekli sistem bağımlılıklarını kurun: 
    sudo apt update
sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-4.0 gir1.2-adw-1 python3-numpy python3-pip python3-venv python3-dev libcairo2-dev build-essential
  4. Terminali, kaynak kod dosyalarını çıkarttığınız klasörde açın.
  5. Sanal ortam oluşturup aktifleştirin: 
    python3 -m venv venv
source venv/bin/activate
  6. Gerekli Python kütüphanesini kurun (diğerleri sistemden geldi): 
    pip install PyMuPDF
  7. Uygulamayı çalıştırın: 
    python3 run-editor.py
